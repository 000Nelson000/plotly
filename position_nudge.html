Nudge points.

Usage

position_nudge(x = 0, y = 0)

Arguments

x, y
Amount of vertical and horizontal distance to move.

Description

This is useful if you want to nudge labels a little ways from their points.

Examples

df <- data.frame(
  x = c(1,3,2,5),
  y = c("a","c","d","c")
)

ggplot(df, aes(x, y)) +
  geom_point() +
  geom_text(aes(label = y))

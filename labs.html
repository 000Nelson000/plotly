Change axis labels and legend titles

Usage

labs(...)


xlab(label)


ylab(label)


ggtitle(label)

Arguments

...
a list of new names in the form aesthetic = "new name"
label
The text for the axis or plot title.

Description

Change axis labels and legend titles

Examples

p <- ggplot(mtcars, aes(mpg, wt)) + geom_point()
